Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. Bootstrap

Bootstrap (formerly Twitter Bootstrap) is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Django

Django is a Python framework that makes it easier to create web sites using Python. Django takes care of the difficult stuff so that you can concentrate on building your web applications. Django emphasizes reusability of components, also referred to as DRY and comes with ready-to-use features like login system, database connection.

MySql

MySQL is an open-source, Relational Database Management System that stores data in a structured format using rows and columns. MySQL stands out as a powerful and versatile open-source Relational Database Management System. Its reliability, scalability, performance, and compatibility make it a preferred choice across various industries.
